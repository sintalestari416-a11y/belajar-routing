# 🚦 Belajar Routing di React

Selamat datang di proyek latihan **React Router** saya! Repositori ini berisi dokumentasi proses belajar saya dalam mengimplementasikan navigasi antar halaman (*routing*) pada aplikasi web berbasis **React**.

## 📋 Tentang Proyek
Proyek ini difokuskan pada penggunaan library **React Router DOM** untuk menciptakan pengalaman navigasi yang mulus (Single Page Application) tanpa perlu memuat ulang halaman browser sepenuhnya.

## 🚀 Fitur & Materi Belajar
* **Navigasi Dasar**: Menggunakan komponen seperti `BrowserRouter`, `Routes`, dan `Route` untuk mengatur alur halaman.
* **Link Navigation**: Implementasi komponen `Link` atau `NavLink` untuk perpindahan antar halaman yang responsif.
* **Manajemen Data**: Mengintegrasikan data dinamis dari sumber eksternal agar tampil sesuai rute yang diakses.
* **Standar Kode**: Penulisan kode tetap rapi dan konsisten dengan bantuan konfigurasi **Prettier**.

## 🛠️ Teknologi yang Digunakan
* **React**: Pustaka inti untuk membangun antarmuka pengguna.
* **React Router DOM**: Library standar untuk menangani routing di React.
* **Vite**: Alat pengembangan (build tool) yang digunakan untuk efisiensi koding.

## 📂 Struktur Penting
* **`index.js`**: Titik masuk utama aplikasi di mana konfigurasi routing biasanya didefinisikan.
* **`data.js`**: Menyediakan dataset pendukung untuk ditampilkan di berbagai rute.
* **`index.html`**: Halaman utama tempat aplikasi React di-*mounting*.
